Native iOS app for Artifact Keeper, built with SwiftUI and targeting iOS 17+.

## Tech Stack

- **SwiftUI** with iOS 17+ APIs
- **Swift 6** with strict concurrency
- **Alamofire** for networking
- **Swift Dependencies** for dependency injection
- **Kingfisher** for image loading

## Design

Follows Apple Human Interface Guidelines with:
- Native navigation patterns (NavigationStack, TabView)
- SF Symbols for iconography
- Dynamic Type support
- Dark/Light mode support
- Haptic feedback

## Features (Subset of Web)

- Dashboard with key metrics
- Repository browsing and search
- Artifact details and downloads
- Security scan results
- Push notifications for scan completion
- Face ID / Touch ID authentication

## Getting Started

Open `Package.swift` in Xcode 16+ and build for iOS Simulator or device.
